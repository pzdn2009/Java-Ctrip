# Live Templates 添加注释

1. create Live Templates Group-- pzdn;
2. create Live Templates -- cc;

   Template Text:

   ```text
   /**
   * @auther zen peng.
   * @version 1.0
   * date-time: $CREAT_DATE$-$CREAT_TIME$
   * desc:
   **/
   ```

3. edit variables

   ```text
   $CREAT_DATE date()
   $CREAT_TIME time()
   ```

4. apply this template to Java.
